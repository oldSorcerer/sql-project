
### Предотвращение деления на ноль с использованием `NULLIF()`

У вас есть таблица `sales` с данными о продажах.

Напишите SQL-запрос, который рассчитывает рентабельность расходов (`revenue / expenses`), используя `NULLIF()` для предотвращения деления на ноль. Если `expenses` равно 0, результат должен быть `NULL`.

#### Требования:
1. Запрос должен использовать функцию `NULLIF()` для предотвращения деления на ноль.
2. Если значение столбца `expenses` равно 0, должно возвращаться `NULL` вместо результата деления.
3. Рентабельность (`profitability`) должна рассчитываться как отношение `revenue / expenses`, если `expenses` не равно 0.
4. Результат расчета рентабельности должен быть представлен в виде числа с двумя знаками после запятой.
