
### Использование HAVING для фильтрации среднего рейтинга

У вас есть таблица `reviews` с данными об отзывах.

Ваша задача:

- Найти продукты, у которых средний рейтинг больше или равен 4.

- Используйте `HAVING` для фильтрации.

- Выведите два столбца: `product` и `avg_rating`, упорядочив по убыванию рейтинга.

#### Требования:
1. Запрос должен вычислять средний рейтинг для каждого продукта с использованием функции `AVG()`.
2. Запрос должен группировать данные по столбцу `product` с помощью оператора `GROUP BY`.
3. Запрос должен использовать оператор `HAVING` для фильтрации продуктов, у которых средний рейтинг больше или равен 4.
4. Запрос должен выводить только два столбца: `product` и `avg_rating`.
5. Результат должен быть отсортирован по столбцу `avg_rating` в порядке убывания.
