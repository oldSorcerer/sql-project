
### Подсчёт строк с заполненными и незаполненными данными

В таблице `products` есть информация о продуктах, включая их цену (`price`).

Напишите SQL-запрос, который возвращает:

1. Общее количество строк в таблице (`total_rows`).

2. Количество строк, где поле `price` не является `NULL` (`non_null_prices`).

3. Количество строк, где поле `price` равно `NULL` (`null_prices`).

Результат должен содержать три столбца: `total_rows`, `non_null_prices`, `null_prices`.

#### Требования:
1. Запрос должен использовать агрегатные функции, такие как `COUNT()`, для подсчета строк.
2. Запрос должен подсчитывать общее количество строк в таблице `products` и возвращать это значение в столбце `total_rows`.
3. Запрос должен подсчитывать количество строк, где поле `price` не равно `NULL`, и возвращать это значение в столбце `non_null_prices`.
4. Запрос должен подсчитывать количество строк, где поле `price` равно `NULL`, и возвращать это значение в столбце `null_prices`.
5. Запрос должен корректно учитывать специфику работы с `NULL` значениями, используя операторы `IS NULL` и `IS NOT NULL`.
