
### Сумма и среднее значение оценок

В таблице `students_scores` хранится информация об оценках студентов.

Напишите SQL-запрос, который вычисляет:

1. Общую сумму оценок.

2. Среднее значение всех оценок.

Результат должен содержать два столбца: `total_score` и `average_score`.

#### Требования:
1. Запрос должен использовать функции `SUM()` и `AVG()` для вычисления общей суммы и среднего значения оценок соответственно.
2. Запрос должен корректно обрабатывать возможные значения `NULL` в столбце с оценками, игнорируя их при вычислении суммы и среднего значения.
3. Результат запроса должен содержать два столбца с именами `total_score` и `average_score`.
