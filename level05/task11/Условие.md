
### Условное отображение отсутствующих данных

В таблице `employees` есть столбцы `name` и `position`. В столбце `position` иногда встречаются `NULL`, если должность не указана.

Напишите SQL-запрос, который выведет столбцы `name` и `position_status`, где:

- Если `position` равен `NULL`, в `position_status` должно быть значение `'Position Not Specified'`.

- Если `position` указан, в `position_status` должно быть значение `'Position Specified'`.

#### Требования:
1. Запрос должен использовать оператор `CASE` для создания условного выражения, которое проверяет значение столбца `position`.
2. Условие в операторе `CASE` должно проверять, является ли значение столбца `position` равным `NULL`.
3. Если значение `position` равно `NULL`, в столбце `position_status` должно отображаться значение `'Position Not Specified'`.
4. Если значение `position` не равно `NULL`, в столбце `position_status` должно отображаться значение `'Position Specified'`.
5. Запрос должен выводить два столбца: `name` и `position_status`.
