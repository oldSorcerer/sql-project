
### Использование `COALESCE()` с `GREATEST()` и `LEAST()`

В таблице `student_scores` хранятся оценки студентов за три экзамена: `exam_1`, `exam_2`, `exam_3`. Если значение оценки отсутствует (`NULL`), это означает, что студент не сдавал экзамен, и оценка должна считаться равной 0.

Напишите SQL-запрос, который:

1. Находит наивысшую оценку для каждого студента, учитывая `NULL` как 0.

2. Находит наименьшую оценку для каждого студента, учитывая `NULL` как 0.

#### Требования:
1. Запрос должен использовать функцию COALESCE для замены значений NULL на 0.
2. Для нахождения наивысшей оценки (highest_score) необходимо использовать функцию GREATEST с учетом преобразованных значений (NULL заменен на 0).
3. Для нахождения наименьшей оценки (lowest_score) необходимо использовать функцию LEAST с учетом преобразованных значений (NULL заменен на 0).
4. Запрос должен учитывать все три экзамена (exam_1, exam_2, exam_3) при вычислении наивысшей и наименьшей оценки.
5. Результаты должны быть рассчитаны для каждого студента отдельно, с идентификатором student_id.
