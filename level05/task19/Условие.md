
### Обработка `NULL` значений при агрегатных функциях

В таблице `employees` содержатся данные о сотрудниках.

Напишите SQL-запрос, который вычисляет:

1. Среднюю зарплату сотрудников с учётом всех значений, включая замену `NULL` на `0`.

2. Среднюю зарплату сотрудников без учёта `NULL`.

#### Требования:
1. Таблица employees должна существовать и включать следующие столбцы: employee_id, name, department, salary.
2. Для расчета средней зарплаты с учетом NULL значений необходимо использовать функцию COALESCE(salary, 0), чтобы заменить NULL на 0.
3. Стандартная агрегатная функция AVG(salary) должна использоваться для расчета средней зарплаты без учета NULL значений.
4. Результат запроса должен содержать два столбца: avg_salary_with_null и avg_salary_without_null.
5. - avg_salary_with_null должен отражать среднюю зарплату с учетом замены NULL на 0. - avg_salary_without_null — среднюю зарплату без учета NULL значений.
6. Оба значения в результирующей таблице должны отображаться с двумя знаками после запятой (например, 45000.00).
