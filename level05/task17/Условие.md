
### Фильтрация строк с NULL значениями

У вас есть таблица `employees`, которая содержит информацию о сотрудниках.

Напишите SQL-запрос, который выбирает всех сотрудников, у которых значение `department_id` равно `NULL`.

#### Требования:
1. Таблица employees должна существовать и содержать следующие столбцы: employee_id, name, age, department_id.
2. Запрос должен отбирать только те строки, где значение в столбце department_id является NULL. Для этого необходимо использовать условие IS NULL.
3. Результирующий запрос должен содержать только два столбца: employee_id и name.
