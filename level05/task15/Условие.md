
### Использование `GREATEST()` и `LEAST()`

В таблице `product_prices` хранятся цены товаров в трёх валютах: `price_usd`, `price_eur`, `price_gbp`. Напишите SQL-запрос, который определяет:

1. Самую высокую цену на товар среди всех валют (`max_price`).

2. Самую низкую цену на товар среди всех валют (`min_price`).

#### Требования:
1. Запрос должен использовать функции `GREATEST()` и `LEAST()` для определения максимальной и минимальной цены среди всех валют для каждого товара.
2. Функции `GREATEST()` и `LEAST()` должны корректно учитывать наличие значений `NULL` в столбцах, возвращая `NULL` в случае, если все сравниваемые значения — `NULL`.
3. Запрос должен возвращать результат для каждого `product_id` из таблицы `product_prices`, даже если у товара есть только значения `NULL` в столбцах цен.
4. Результат должен содержать три столбца: `product_id`, `max_price` (максимальная цена) и `min_price` (минимальная цена).
