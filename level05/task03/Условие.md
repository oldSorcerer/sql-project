
### Поиск записей с отсутствующим значением

В базе данных есть таблица `students`, содержащая информацию о студентах.

Напишите SQL-запрос, который извлечет всех студентов без указанного email.

#### Требования:
1. Запрос должен быть выполнен к таблице `students`, содержащей указанные столбцы (`student_id`, `name`, `email`, `birth_date`).
2. Запрос должен использовать конструкцию `IS NULL` для проверки отсутствия значения в столбце `email`.
3. Запрос должен извлекать только столбцы `student_id` и `name` из таблицы `students`.
4. В результат должны попадать только те записи, у которых значение в столбце `email` равно `NULL`.
