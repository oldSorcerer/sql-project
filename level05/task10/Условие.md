
### Фильтрация заказов со скидкой с помощью `NULLIF()`

У вас есть таблица `orders` с данными о заказах.

Напишите запрос, который выбирает строки, где `original_price` не равен `discounted_price`, используя `NULLIF()`. Если значения равны, такие строки не должны попадать в результат.

#### Требования:
1. Запрос должен использовать функцию `NULLIF()` для сравнения значений столбцов `original_price` и `discounted_price`.
2. Только строки, где `original_price` не равен `discounted_price`, должны быть включены в результат. Если значения равны, строка должна быть исключена из результата.
3. Запрос должен быть выполнен над таблицей `orders`, содержащей указанные столбцы (`order_id`, `original_price`, `discounted_price`).
