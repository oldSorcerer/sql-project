
### Замена NULL значений с несколькими уровнями приоритетов

У вас есть таблица `contacts` с данными о пользователях.

Напишите SQL-запрос, который создает новый столбец с названием `preferred_email`, содержащий:

- `primary_email`, если он не равен NULL,

- в случае, если `primary_email` равен NULL, использовать `secondary_email`,

- если оба равны NULL, использовать `backup_email`,

- если все значения NULL, то вернуть слово "Нет email".

Выведите столбцы `contact_id` и `preferred_email`.

#### Требования:
1. Запрос должен использовать функцию COALESCE для замены значений NULL на основе приоритетов.
2. Запрос должен следовать заданному приоритету: сначала использовать `primary_email`, затем `secondary_email`, затем `backup_email`.
3. Если все три столбца (`primary_email`, `secondary_email`, `backup_email`) равны NULL, запрос должен вернуть строку "Нет email".
4. Запрос должен возвращать только столбцы `contact_id` и новый столбец `preferred_email`.
