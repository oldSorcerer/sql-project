
### Замена NULL значений на значение по умолчанию

У вас есть таблица `employees` с данными о сотрудниках.

Напишите SQL-запрос, который заменяет все значения `NULL` в столбце `department_id` на значение `0`. Запрос должен вернуть `employee_id`, `name` и новый столбец с названием `resolved_department_id`.

#### Требования:
1. Запрос должен использовать функцию `COALESCE()` для замены значений `NULL` на значение по умолчанию (в данном случае `0`).
2. Результат запроса должен содержать новый столбец с названием `resolved_department_id`, в котором будут отображаться значения из столбца `department_id` с заменой `NULL` на `0`.
3. Запрос должен возвращать оригинальные значения столбцов `employee_id` и `name` без изменений.
4. Результат запроса должен содержать три столбца: `employee_id`, `name`, и `resolved_department_id`.
