
### Присвоение текстовых описаний оценкам

В базе данных есть таблица `students`, содержащая информацию о студентах.

Напишите SQL-запрос, который добавляет текстовое описание оценок. Используйте `CASE`. Столбец с описанием должен называться `grade_description` и содержать:

- "Отлично" для оценки 5,

- "Хорошо" для оценки 4,

- "Удовлетворительно" для оценки 3,

- "Неудовлетворительно" для оценки 2,

- "Не указано" для значений `NULL` в поле `grade`.

#### Требования:
1. Запрос должен использовать конструкцию `CASE` для присвоения текстовых описаний оценкам в зависимости от их значений.
2. Запрос должен корректно обрабатывать значения `NULL` в поле `grade`, присваивая текстовое описание "Не указано".
3. Результат запроса должен содержать новый столбец `grade_description`, в котором будут текстовые описания оценок.
4. Каждой числовой оценке (2, 3, 4, 5) должно быть присвоено соответствующее текстовое описание ("Неудовлетворительно", "Удовлетворительно", "Хорошо", "Отлично").
