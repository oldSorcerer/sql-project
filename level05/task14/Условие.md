
### Сравнение категорий товаров

В таблице `products` хранится информация о продуктах.

Напишите SQL-запрос, который переводит коды категорий в понятные названия. Используйте `CASE`. Новый столбец должен называться `category_name` и содержать:

- "Электроника" для `el`,

- "Мебель" для `frn`,

- "Книги" для `bks`,

- "Не указано" для значений `NULL`.

#### Требования:
1. Запрос должен использовать конструкцию `CASE` для преобразования значений столбца `category_code` в понятные названия.
2. Для значений `category_code`, равных `NULL`, в столбце `category_name` должно быть указано "Не указано".
3. Значения `el`, `frn` и `bks` в столбце `category_code` должны быть преобразованы в "Электроника", "Мебель" и "Книги" соответственно.
4. Новый столбец в результирующем запросе должен называться `category_name`.
