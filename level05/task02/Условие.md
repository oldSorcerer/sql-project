
### Подсчёт клиентов по регионам с фильтрацией

У вас есть таблица `customers`с данными о клиентах.

Ваша задача:

- Посчитать количество клиентов в каждом регионе.

- Вывести только те регионы, где клиентов больше двух.

- Результат отсортируйте по количеству клиентов по убыванию.

#### Требования:
1. Запрос должен использовать функцию `COUNT()` для подсчета количества клиентов в каждом регионе.
2. Запрос должен содержать оператор `GROUP BY` для группировки данных по столбцу `region`.
3. Должен быть применен оператор `HAVING` для фильтрации групп, где количество клиентов больше двух.
4. Результат должен быть отсортирован по количеству клиентов (`customer_count`) в порядке убывания с помощью `ORDER BY`.
