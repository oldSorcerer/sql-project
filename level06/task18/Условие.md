
### Самообъединение для поиска связанных товаров

У вас есть таблица `products` с данными продуктов.

Напишите SQL-запрос, который с использованием `SELF JOIN` найдет пары товаров из одной категории.

Каждая пара должна отображаться только один раз (например, `Холодильник — Стиральная машина`, но не наоборот).

#### Требования:
1. Запрос должен использовать `SELF JOIN` для объединения таблицы `products` самой с собой.
2. Условие объединения должно проверять, что товары находятся в одной и той же категории.
3. Запрос должен гарантировать, что каждая пара товаров отображается только один раз (например, `Холодильник — Стиральная машина`, но не наоборот).
4. Условие объединения должно исключать случаи, когда оба товара в паре являются одним и тем же товаром.
5. Результат запроса должен содержать только два столбца — `product_1` и `product_2`, представляющие названия товаров.
6. Для предотвращения дублей пары должны быть упорядочены, чтобы `product_1` всегда был лексикографически меньше, чем `product_2`.
