
### Выбор записей за текущий год

В таблице `employees` содержатся данные о сотрудниках: `id`, `name`.

В таблице `attendances` указаны их посещения за разные годы: `employee_id`, `attendance_date`.

Напишите SQL-запрос, который вернёт список сотрудников, которые имели посещения в текущем году.

Используйте фильтрацию по году в разделе `ON` при объединении таблиц.

#### Требования:
1. Таблицы `employees` и `attendances` должны быть связаны через внешний ключ `employee_id` в таблице `attendances`, ссылающийся на `id` в таблице `employees`.
2. При объединении таблиц с помощью `JOIN` необходимо использовать условие фильтрации по году (`YEAR(attendance_date) = YEAR(CURRENT_DATE)`).
3. Результат запроса должен содержать уникальные имена сотрудников (`name`) из таблицы `employees`, которые соответствуют заданному условию.
4. Для корректного результата необходимо использовать `INNER JOIN`, чтобы выбрать только тех сотрудников, у которых есть записи о посещениях.
