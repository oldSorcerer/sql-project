
### Базовый запрос с множественным JOIN

У вас есть три таблицы: `students` (студенты), `courses` (курсы) и `enrollments` (записи на курсы).

Напишите SQL-запрос, который объединяет эти три таблицы и возвращает список студентов с их курсами. Результирующая таблица должна включать следующие столбцы:

- `student_name` (имя студента).

- `course_name` (название курса).

#### Требования:
1. Запрос должен включать объединение трех таблиц (`students`, `courses`, `enrollments`) с использованием `INNER JOIN` для связывания данных.
2. Таблица `enrollments` должна быть связана с таблицей `students` по полю `student_id` и с таблицей `courses` по полю `course_id`.
3. Запрос должен возвращать только два столбца — `student_name` (из таблицы `students`) и `course_name` (из таблицы `courses`).
4. Каждая строка результата должна соответствовать уникальной паре студент-курс, как указано в таблице `enrollments`.
