
### Сравнение студентов и курсов используя FULL OUTER JOIN

Даны две таблицы: `students` и `courses`.

Напишите SQL-запрос, который выведет полный список студентов и курсов, с учетом студентов без курсов и курсов без студентов. Выведите столбцы: `student_id`, `name`, `course_name`.

#### Требования:
1. Запрос должен использовать оператор FULL OUTER JOIN для объединения двух таблиц `students` и `courses`, чтобы включить все строки из обеих таблиц, даже если для них нет соответствия.
2. Результат должен включать студентов, у которых нет соответствующих курсов, и курсы, у которых нет соответствующих студентов.
3. Запрос должен выводить три столбца: `student_id`, `name` и `course_name`.
4. Для строк, где нет соответствия между таблицами, значения должны быть представлены как NULL.
5. Объединение таблиц должно быть выполнено по столбцу `student_id`, который является общим для обеих таблиц.
6. Столбцы в результате должны быть упорядочены в следующем порядке: `student_id`, `name`, `course_name`.
