
### Использование `RIGHT JOIN` с фильтрацией

У вас есть таблицы: `employees` — список сотрудников, `departments` — список отделов.

Напишите SQL-запрос с использованием `RIGHT JOIN`, чтобы получить список всех отделов и связанных с ними сотрудников. Добавьте условие, чтобы вернуть только те отделы, в которых нет сотрудников.

Выведите столбцы: `department_name`, `name`.

#### Требования:
1. Запрос должен использовать оператор `RIGHT JOIN` для объединения таблиц `departments` и `employees`.
2. Запрос должен включать все записи из таблицы `departments`, даже если в таблице `employees` нет соответствующих записей.
3. Запрос должен содержать условие для фильтрации только тех отделов, у которых нет связанных сотрудников.
4. Запрос должен возвращать только два столбца: `department_name` и `name`.
5. Запрос должен корректно обрабатывать значения `NULL`, чтобы определить отделы без сотрудников.
