
### Список покупателей и их заказов

У вас есть две таблицы:

1. `customers` — таблица с данными о клиентах.

2. `orders` — таблица с данными о заказах.

Напишите SQL-запрос, который возвращает все заказы с указанием имени клиента и суммы заказа. Используйте `INNER JOIN` для объединения таблиц на основе `customer_id`.

#### Требования:
1. Запрос должен использовать оператор `INNER JOIN` для объединения таблиц `customers` и `orders` на основе общего столбца `customer_id`.
2. Таблицы `customers` и `orders` должны быть объединены по совпадающим значениям в столбце `customer_id`.
3. Результат запроса должен содержать три столбца: `order_id` из таблицы `orders`, `name` из таблицы `customers` и `amount` из таблицы `orders`.
