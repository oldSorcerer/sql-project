
### Выбор студентов с активными курсами

В таблице `students` находятся данные о студентах с колонками `id` и `name`.

В таблице `enrollments` — информация об их курсах: `student_id`, `course_id`, `status`.

В таблице `courses` — информация о курсах: `id` и `name`.

Напишите SQL-запрос, который вернёт имена студентов и названия их курсов, где статус участия равен `'active'`.

Используйте `JOIN` для объединения таблиц и добавьте фильтрацию по статусу внутри условия `ON`.

#### Требования:
1. Запрос должен использовать `JOIN` для объединения таблиц `students`, `enrollments` и `courses` на основе внешних ключей.
2. Фильтрация записей по статусу `'active'` должна быть выполнена в условии `ON` при объединении таблиц.
3. Запрос должен возвращать два столбца: `student_name` (имя студента из таблицы `students`) и `course_name` (название курса из таблицы `courses`).
4. Связь между таблицами должна быть установлена следующим образом: - `students.id` = `enrollments.student_id` - `courses.id` = `enrollments.course_id`.
5. В итоговый результат должны попасть только те записи, где статус в таблице `enrollments` равен `'active'`.
