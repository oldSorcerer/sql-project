
### Подсчёт студентов без курсов (с использованием `COUNT`)

У вас есть две таблицы: `students` (студенты) и `enrollments` (записи на курсы).

Напишите запрос, который подсчитывает, сколько студентов не записано ни на один курс (их записи в таблице `enrollments` отсутствуют).

#### Требования:
1. Запрос должен использовать `LEFT JOIN` для объединения таблицы `students` с таблицей `enrollments`.
2. После объединения необходимо проверить, где значения в столбце `course_name` из таблицы `enrollments` равны `NULL`.
3. Запрос должен подсчитывать количество строк, соответствующих условию отсутствия записей в таблице `enrollments`, с помощью агрегатной функции `COUNT`.
4. Предполагается, что столбец `id` в таблице `students` является уникальным идентификатором для каждого студента.
5. При написании запроса должны использоваться алиасы для упрощения чтения и понимания кода.
