
### Использование COALESCE для обработки NULL в строках

В таблице `projects` содержатся данные о проектах.

Сформируйте запрос, который:

1. Заменяет `NULL` в столбце `manager` на строку `'Не назначен'`

2. Сортирует проекты по алфавиту `project_name`

#### Требования:
1. Запрос должен использовать функцию COALESCE для обработки значений NULL в столбце `manager`, заменяя их на строку `'Не назначен'`.
2. Результат запроса должен быть отсортирован в алфавитном порядке по столбцу `project_name`.
3. Запрос должен включать все столбцы таблицы `projects` (`project_id`, `project_name`, `manager`) в результирующем наборе.
4. Значения NULL в столбце `manager` должны быть заменены на строку `'Не назначен'` без изменения исходных данных в таблице.
