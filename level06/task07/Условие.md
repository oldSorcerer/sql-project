
### Простое использование `RIGHT JOIN`

У вас есть две таблицы: `products` — список продуктов, `sales` — список продаж продуктов.

Напишите SQL-запрос с использованием `RIGHT JOIN`, чтобы получить список всех продаж и соответствующих названий продуктов. Если продукт не найден, в столбце названия должно быть `NULL`.

Выведите следующие столбцы: `sale_id`, `product_id`, `name`, `quantity`.

#### Требования:
1. Запрос должен использовать оператор `RIGHT JOIN` для объединения таблиц `products` и `sales`.
2. Результат должен содержать все строки из таблицы `sales`, даже если для них нет соответствия в таблице `products`.
3. Таблицы должны быть объединены по столбцу `product_id`.
4. Если для строки из таблицы `sales` нет соответствующего продукта в таблице `products`, столбец `name` должен содержать значение `NULL`.
5. Запрос должен возвращать именно указанные столбцы: `sale_id`, `product_id`, `name`, `quantity`.
