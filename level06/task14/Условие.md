
### Фильтрация данных с использованием WHERE и множественных JOIN

У вас есть три таблицы: `students` (студенты), `courses` (курсы) и `enrollments` (записи на курсы).

Напишите SQL-запрос, который выведит имена студентов и названия их курсов, но только для тех курсов, на которых обучается "Иван Иванов".

Запрос должен вернуть два столбца:

- `student_name` (имя студента).

- `course_name` (название курса).

#### Требования:
1. Запрос должен использовать несколько `JOIN`, чтобы объединить данные из таблиц `students`, `courses` и `enrollments`.
2. В запросе необходимо использовать условие `WHERE`, чтобы выбрать только те записи, где имя студента равно "Иван Иванов".
3. Результат запроса должен содержать только два столбца: `student_name` (имя студента) и `course_name` (название курса).
4. Должны быть правильно установлены связи между таблицами: - `students.id` = `enrollments.student_id` - `courses.id` = `enrollments.course_id`.
5. Запрос должен возвращать все курсы, на которых обучается "Иван Иванов", без пропуска записей.
