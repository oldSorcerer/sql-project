
### Использование FULL OUTER JOIN для объединения данных

У вас есть две таблицы: `authors` и `books`.

Напишите SQL-запрос, который с использованием `FULL OUTER JOIN` выведет все данные из обеих таблиц, включая тех авторов, которые не написали книги, и книги, которые не имеют авторов в таблице `authors`.

Результат должен содержать столбцы: `author_id`, `name`, `title`.

#### Требования:
1. Запрос должен использовать оператор `FULL OUTER JOIN` для объединения данных из таблиц `authors` и `books`.
2. Результат запроса должен включать все строки из обеих таблиц, независимо от наличия соответствующих данных в другой таблице.
3. Объединение таблиц должно выполняться на основании столбца `author_id`, который является общим для таблиц `authors` и `books`.
4. Если у книги нет автора в таблице `authors`, то в столбце `name` должно быть значение `NULL`. Если у автора нет книги в таблице `books`, то в столбце `title` должно быть значение `NULL`.
5. Результат должен содержать только три столбца: `author_id`, `name`, `title`.
