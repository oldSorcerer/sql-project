
### Полный список студентов и их курсов

У вас есть две таблицы:

1. Таблица `students` содержит информацию о студентах.

2. Таблица `enrollments` содержит информацию о том, какие студенты записаны на курсы.

Напишите SQL-запрос, который выведет полный список студентов и курсов. Для студентов, которые не записаны на курсы, в столбце курса должно быть значение `NULL`.

#### Требования:
1. Запрос должен использовать оператор `LEFT JOIN` для объединения таблиц `students` и `enrollments`, чтобы включить всех студентов, даже если они не записаны на курсы.
2. Результат должен включать записи всех студентов из таблицы `students`, независимо от наличия соответствующих записей в таблице `enrollments`.
3. Если студент не записан на курс, в столбце `course` должно быть указано значение `NULL`.
4. Запрос должен выводить только столбцы `name` (имя студента) из таблицы `students` и `course` (название курса) из таблицы `enrollments`.
