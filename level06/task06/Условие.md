
### Полный список продуктов и их продаж

У вас есть две таблицы:

1. Таблица `products`, содержащая информацию о всех продуктах.

2. Таблица `sales`, содержащая данные о продажах.

Напишите SQL-запрос, который выведет полный список всех продуктов и количество их продаж. Если продукт еще не был продан, в столбце количества продаж должно быть значение `NULL`.

#### Требования:
1. Запрос должен использовать `LEFT JOIN` для объединения таблиц `products` и `sales`, чтобы включить все строки из таблицы `products`, даже если соответствующих данных в таблице `sales` нет.
2. Для подсчета общего количества продаж для каждого продукта необходимо использовать группировку (`GROUP BY`) по полю `product_id` из таблицы `products`.
3. Для подсчета общего количества продаж необходимо использовать агрегатную функцию `SUM()` для столбца `quantity` из таблицы `sales`.
4. Если для продукта нет данных о продажах в таблице `sales`, значение в столбце `total_sold` должно быть `NULL`.
5. Результат запроса должен содержать только два столбца: `product_name` из таблицы `products` и вычисляемый столбец `total_sold`.
