
### Простое объединение студентов и курсов

У вас есть две таблицы:

1. `students` — содержит информацию о студентах.

2. `enrollments` — содержит информацию о записи студентов на курсы.

Напишите SQL-запрос, который возвращает имена студентов и идентификаторы курсов, на которые они записаны. Используйте `INNER JOIN` для объединения таблиц на основе `student_id`.

#### Требования:
1. Запрос должен использовать оператор `INNER JOIN` для объединения таблиц `students` и `enrollments` на основе совпадающих значений в столбце `student_id`.
2. Результат запроса должен содержать только два столбца: `name` из таблицы `students` и `course_id` из таблицы `enrollments`.
3. Объединение таблиц должно быть выполнено по столбцу `student_id`, который является общим для обеих таблиц.
