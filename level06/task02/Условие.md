
### Расчёт средней цены с условием и обработкой NULL

В таблице `items` содержатся данные о товарах.

Напишите SQL-запрос, который:

1. Выводит `category` и среднюю цену (`avg_price`) только по тем записям, где цена не NULL

2. Оставляет только те категории, где средняя цена больше 500

#### Требования:
1. Запрос должен учитывать только те записи, где значение в столбце `price` не равно NULL.
2. Данные должны быть сгруппированы по столбцу `category` для расчёта средней цены.
3. Для расчёта средней цены необходимо использовать функцию `AVG()`.
4. В запросе нужно применить условие для отбора только тех категорий, у которых средняя цена (`avg_price`) больше 500.
5. Результат должен содержать только два столбца: `category` и `avg_price`.
